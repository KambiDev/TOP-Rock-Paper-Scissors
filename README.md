# TOP-Rock-Paper-Scissors

A classic game implementation developed in JavaScript.

## Description
This project allows the user to compete against the computer in a 5-round match. Interaction is handled entirely through browser pop-ups (`prompt` for inputs and `alert` for results), making it a fully interactive experience.

## Features
- **Interactive UI:** Uses `prompt` and `alert` for gameplay.
- **Game Logic:** Automates a 5-round match loop.
- **Input Validation:** Case-insensitive inputs.
- **Score Tracking:** Tracks wins and declares a final winner after 5 rounds.

## Technologies
- JavaScript (ES6)
